## 규칙

1. 일주일에 두 번 팀원들과 일정을 맞추어 진행합니다.
2. 미참여시 불이익은 없으나 다른 팀원들과 진도를 위해 문제를 풀어서 와 주세요.
3. 해당 문제에 대한 폴더가 생성되면 본인의 풀이를 pr로 보내 주세요.
